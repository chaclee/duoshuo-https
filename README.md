# duoshuo-https
让多说评论头像,表情支持https

**原理** :将https图片转由七牛CDN处理

**使用方法**
- 方法一:
```
<script src="https://dn-hb0716.qbox.me/duoshuo.js"></script>
```
- 方法二:
下载`duoshuo.js`引用到你的网站文件
